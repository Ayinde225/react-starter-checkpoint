# React Starter Checkpoint

A Create React App baseline scaffolded as a React JS checkpoint. It provides the standard CRA structure (`App`, `index`, testing setup, web-vitals reporting) as a clean starting point for React exercises.

## Features

- Standard Create React App project structure
- Ready for `npm start` / `npm test` / `npm run build`
- Jest + React Testing Library configured out of the box

## Getting started

```bash
git clone https://github.com/Ayinde225/react-starter-checkpoint.git
cd react-starter-checkpoint
npm install
npm start
```

Then open http://localhost:3000 in your browser.
